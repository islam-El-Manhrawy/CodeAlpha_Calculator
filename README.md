# Calculator Program - C Language

A simple calculator program built in C as part of CodeAlpha Internship.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Continuous operation until user exits

## How to Run
1. Compile: gcc calculator.c -o calculator
2. Run: ./calculator

## Built With
- C Programming Language
- GCC Compiler
